  <files>src/hooks/useLeaderboard.ts</files>
  <behavior>
    - Hook initializes with isLoading=true, items=[], error=null
    - On mount, selects 1 random seed from each niche (tech, design, project, devops)
    - Fires 4 parallel fetch calls to `/api/malt/autocomplete?q={seed}`
    - All 4 requests complete in parallel (not sequential)
    - Returns ranked items: rank (1-4) + first suggestion from each response
    - On success: isLoading=false, error=null, items populated
    - On fetch failure: isLoading=false, error set to error message, items=[]
    - Cleanup on unmount (no memory leaks from pending requests)
  </behavior>
  <action>
    Create src/hooks/useLeaderboard.ts implementing Pattern 1 from RESEARCH.md (lines 168-233).

    Hook signature:
    ```typescript
    export function useLeaderboard() {
      return {
        items: Array<{ rank: number; suggestion: MaltSuggestion }>,
        isLoading: boolean,
        error: string | null
      }
    }
    ```

    Implementation details (per RESEARCH.md):
    - Define NICHE_SEEDS object with 4-5 seed terms per niche
    - useEffect hook fires once on mount (empty dependency array)
    - Select random seed from each niche using Math.random()
    - Create fetcher function accepting query string
    - Use Promise.all() to fire all 4 fetches concurrently
    - Map responses to ranked items (index + 1 = rank number)
    - Handle errors with .catch() and set error state
    - Finally block always sets isLoading=false

    Do NOT use SWR for leaderboard (different pattern from useSearch). Per D-08 and RESEARCH.md line 747, leaderboard should fetch fresh every page load without caching.

    Example structure:
    ```typescript
    const NICHE_SEEDS = {
      tech: ["React", "Python", "Node.js", "TypeScript", "Go"],
      design: ["Figma", "UX Design", "Webflow", "Branding", "Adobe XD"],
      project: ["Product Manager", "Scrum Master", "Agile", "Project Management"],
      devops: ["Docker", "Kubernetes", "AWS", "CI/CD", "Terraform"]
    };

    export function useLeaderboard() {
      const [items, setItems] = useState([]);
      const [isLoading, setIsLoading] = useState(true);
      const [error, setError] = useState<string | null>(null);

      useEffect(() => {
        const selectedSeeds = Object.entries(NICHE_SEEDS).map(
          ([, seeds]) => seeds[Math.floor(Math.random() * seeds.length)]
        );

        const fetcher = (q: string) =>
          fetch(`/api/malt/autocomplete?q=${encodeURIComponent(q)}`).then((r) => {
            if (!r.ok) throw new Error("Failed to fetch");
            return r.json();
          });

        Promise.all(selectedSeeds.map(fetcher))
          .then((responses) => {
            const ranked = responses.map((resp, idx) => ({
              rank: idx + 1,
              suggestion: resp.suggestions[0] || { label: "N/A", occurrences: 0 }
            }));
            setItems(ranked);
            setError(null);
          })
          .catch((err) => {
            setError(err.message);
          })
          .finally(() => setIsLoading(false));
      }, []);

      return { items, isLoading, error };
    }
    ```
