## Color

| --------------- | ------------------ | ---------------------------------------- |

**Accent reserved for:** Primary CTA ("Search" button), active navigation states, focus indicators

- Rare (<10 users): #10B981 (emerald-500) — green badge
- Common (10–100 users): #F59E0B (amber-500) — yellow badge
- Oversaturated (>100 users): #EF4444 (red-500) — red badge

## Component Specifications

### SearchInput

- **Type:** Input + Search icon (lucide-react)
- **Debounce:** 300ms client-side (prevents excessive API calls)
- **Min height:** 44px (touch target accessibility)
- **Placeholder:** "Ex : développeur React, UX designer..."
- **Icon position:** Right-aligned (standard search convention)

### KeywordCard

- **Displays:** Keyword label, volume count ("N utilisateurs Malt"), competition badge
- **Layout:** Flex row with keyword label | volume | badge
- **Background:** Secondary (gray-100) or white
- **Padding:** 16px (md spacing token)

### CompetitionBadge

- **Type:** Color-coded pill/badge
- **Colors:**
  - Rare: #10B981 (emerald-500)
  - Common: #F59E0B (amber-500)
  - Oversaturated: #EF4444 (red-500)
- **Text:** Short label ("Rare", "Common", "Saturated") or count only
- **Size:** Label/Small typography (14px, 500 weight)

### LoadingSpinner

- **Type:** shadcn Skeleton component
- **Animation:** Fade pulse (standard shadcn pattern)
- **Usage:** Display while search results load

### ErrorMessage

- **Type:** shadcn Alert with destructive variant
- **Copy:** "Search temporarily unavailable. Please try again."
- **Dismissible:** No (persistent until retry succeeds)
